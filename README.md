# treehacks starter

## Structure
- `frontend/` static frontend (HTML/CSS/JS)
- `backend/` Node HTTP backend

## Backend
1. `cd backend`
2. `node server.js`

Optional: set `PORT` via `.env`.

## Frontend
Open `frontend/index.html` in your browser. The frontend expects the backend at `http://localhost:4000`.

## Next steps
- Replace the static UI with your app screens.
- Add real API routes in `backend/server.js`.
- If you want a build step, swap the frontend to Vite/Next/etc.
